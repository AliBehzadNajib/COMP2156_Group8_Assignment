This group project relates to the COMP 2156 curriculum, DevOps for System
Administration. COMP 2156 is an instructional course going through the procedures,
protocols, and best practices for a DevOps engineer career. 

The course aims to teach students how to effectively carry out the roles of a 
System or Network engineer with DevOps processes. Learners will also be able to
demonstrate strategies and skills of using DevOps tools, Cloud database technologies
and techniques for software/network applications.
